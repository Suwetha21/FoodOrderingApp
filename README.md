# FoodOrderingSystem

## Overview
FoodOrderingSystem is a simple Java application designed to simulate a food ordering process for restaurants or cafes. It allows users to view a menu, place orders, and receive an order summary.

## Features
- Display menu items
- Place food orders
- View order summary
- Simple command-line interface

## Prerequisites
- Java Development Kit (JDK) 8 or higher

## How to Compile and Run
1. Open a command prompt and navigate to the project directory.
2. Compile the Java file:
   ```
   javac FoodOrderingSystem.java
   ```
3. Run the program:
   ```
   java FoodOrderingSystem
   ```

## Example Usage
```
Welcome to the Food Ordering System!
1. Pizza - $10
2. Burger - $7
3. Salad - $5
Enter the number of the item you want to order: 1
Enter quantity: 2
Order Summary:
Pizza x2 - $20
Thank you for your order!
```
